Liteseed 笔记法：用五个文件夹终结分类焦虑，让 Claude 成为你的第二大脑管家

Five Folders Architecture
Inbox
纯粹的捕获区。随机想法、网页剪藏、语音备忘录全部倾倒于此
零分类压力 → AI 自动处理
Notes
系统的绝对核心。无子文件夹的扁平目录，所有笔记的唯一安息地
900 或 9000 文件，同一层级
Categories
导航层。容器笔记，动态展示 Notes 中的文件
回答「这是什么类型的笔记」
Subjects
第二导航层。串联跨越类别的核心主题线索
回答「这是关于什么的」
System
基础设施区：模板、附件、仪表盘
维护系统所需的一切
❌ 传统思维陷阱
每次创建笔记，都要缴纳「隐性税负」：这是项目还是资源？属于哪个领域？放在哪个子文件夹？

检索时还要缴纳第二重税：不仅要记住写了什么，还要记住藏在哪里。
✓ Liteseed 范式转变
停止思考「笔记应该放在哪里」，开始追问「这条笔记是什么」

笔记不属于某个物理位置。你描述它，系统找到它；有了 AI，连描述都被自动化了。
